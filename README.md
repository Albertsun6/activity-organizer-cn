# 活动管理工具（单页版）

纯前端、本地存储的单场活动管理小工具，支持报名、候补、签到、导出及 .ics 日历生成。无需后端，直接打开 `index.html` 即可使用。

## 功能
- 活动信息：名称 / 时间 / 地点 / 名额 / 说明
- 报名与候补：超出名额自动进入候补，空位可一键补位
- 签到：手动签到或扫描个人二维码（`?checkin=报名ID` 自动签到）
- 导出：CSV 名单、JSON 数据；支持 JSON 导入
- 日历：一键生成 `.ics`，可导入日历应用
- 分享：复制中文分享文案

## 使用
1) 直接双击 `index.html` 用浏览器打开（Chrome/Edge/Safari 均可）。
2) 先填写“基本设置”，再录入报名信息。
3) 行操作可签到、补位、取消、查看签到二维码。
4) 导入/导出用于备份或换机迁移（本地存储不跨设备）。

## 开发
仅依赖 `qrcodejs` CDN。如需本地预览，可使用任意静态服务器：

```bash
npx serve .
```
